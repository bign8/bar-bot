# Bar-Bot (v0.0.0)

## File structure

* `design/` - All files pertaining to the physical design of the robot
* `www/` - Web based files for application (use gh-pages?)
* `plc/` - Programmable Logic Controller - controll software for robot

## Control Hardware 

* Raspberry-PI
* Stepper Motors + Drivers
* Power Supplies
* Solenoids
* etc
